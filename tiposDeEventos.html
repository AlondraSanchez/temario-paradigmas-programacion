3.1.3 Tipos de eventos y notificaciones

¿Qué es un evento?

Un evento es una accion que es reconocida por un objeto y que normalmente es provocada por el usuario al interactuar con la interfaz del programa. Es una condición que surge durante la ejecución de un programa y que requiere alguna acción por parte del sistema. Cada evento es diferente por naturaleza, algunos requieren que el programa recobre y muestre cierta información, y otros que se inicien algunos cálculos y cambios de estado. Los eventos incluyen al mouse, al teclado, una interfaz de usuario y las acciones que se deben activar en el programa cuando ocurran. Esto significa que el usuario debe interactuar con un objeto en el programa, como hacer clic en un botón del mouse, usar el teclado para seleccionar un botón, etc.

Existen múltiples tipos de eventos, pero los principales son los eventos que se activan con el mouse (click, doble click, etc.), eventos que se activan según alguna circunsancia (como el focus, scroll, arrastrar, soltar, etc.) o eventos que se activan con entradas del teclado del usuario (teclas presionadas o combinaciones de estas).

Para el caso de eventos del teclado, los lenguajes de programación cuentan con métodos para poder detectar el tipo de tecla que se está presionando, es decir si se está presionando una tecla alfanumérica, tecla espacio, tecla enter, tecla alt, tecla ctrl, etc.

A continuación se explica cómo se activan algunos eventos

Evento Click

El evento click se activa al hacer click sobre un elemento. Generalmente este evento está asociado con botones.


Además del Evento click existe el evento doble click. Intenta hacer doble click sobre este texto para activar el evento.

Evento Focus

El evento focus se activa cuando colocas el cursor está apuntando hacia un elemento. Por lo general este evento es aplicado a campos de texto. Este evento tiene su contraparte que se activa cuando el cursor ya no apunta en el elemento.

Eventos del teclado

Los eventos del teclado se dan cuando el usuario presiona teclas en el teclado. Existe un evento para detectar cuando una tecla se presiona, y un evento para detectar cuando una tecla se deja de presionar.
